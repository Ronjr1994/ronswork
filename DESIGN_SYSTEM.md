# Ron's Work Design System

## Brand idea

**Clear systems for work that cannot afford to be messy.**

The visual system should feel editorial, operational, direct, and calm rather than decorative or corporate.

## Foundations

| Element | Standard |
|---|---|
| Background | Near-black `#0a0a09` |
| Surface | Warm cream `#f5f2ec` |
| Accent | Acid lime `#d8ff3e` |
| Display type | Bebas Neue |
| UI/body type | DM Sans |
| Headings | Manrope |
| Corners | Sharp / zero radius |
| Strokes | Thin, visible, restrained |
| Motion | Scroll-linked but stable at the end state |
| Images | Monochrome, high contrast, intentional crop |

## Spacing

Use a restrained 4/8-based rhythm:

- 8–12 px: icon and micro gaps
- 16–24 px: component spacing
- 32–48 px: group spacing
- 72–112 px: section spacing

## Responsive rules

- **Desktop:** full editorial composition and three-column hero end state
- **Laptop:** preserve hierarchy while reducing portrait and type scale
- **Tablet:** two-column systems become one or two columns depending on density
- **Mobile:** stack the hero as `RON'S / portrait / WORK`; keep CTAs full-width when needed
- **Small phone:** reduce type and logo dimensions without hiding core content

## Interaction standards

- Buttons remain rectangular and clearly stroked.
- Hover is secondary; focus-visible states are mandatory.
- Reduced-motion users receive a stable final layout.
- Dialogs trap focus and close with Escape.
- The project assistant never blocks access to the main contact options.
